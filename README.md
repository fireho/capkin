Capkin
======


* Upload your `Android Apps`  to `Google Play`
* Uploads your apks to Alpha, Beta or Production


## Install


    gem install capkin

## Use

Inside your project folder:


```
capkin production
capkin beta
capkin alpha
```


## Capkin

Config file:

```
name: 'Foo'
app: 'com.your.app'
build: 'build/'
```
